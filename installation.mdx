# Installation

Cardio is published to Maven Central and can be added to your project using Gradle or Maven.

## Requirements

Before installing Cardio, ensure your project meets these requirements:

- **Java 21+** - Cardio requires Java 21 or later
- **Kotlin** - Designed for Kotlin projects with coroutines support
- **PostgreSQL** - Cardio works exclusively with PostgreSQL databases

## Add Dependency

Choose your build tool and add the Cardio dependency:

<CodeGroup>
```kotlin Gradle (Kotlin DSL)
dependencies {
    implementation("io.github.blad3mak3r.cardio:cardio-postgres:VERSION")
}
```

```groovy Gradle (Groovy)
dependencies {
    implementation 'io.github.blad3mak3r.cardio:cardio-postgres:VERSION'
}
```

```toml Gradle (Version Catalog)
# In your libs.versions.toml file:
[versions]
cardio = "VERSION"

[libraries]
cardio-postgres = { module = "io.github.blad3mak3r.cardio:cardio-postgres", version.ref = "cardio" }

# Then in your build.gradle.kts:
# dependencies {
#     implementation(libs.cardio.postgres)
# }
```

```xml Maven
<dependency>
    <groupId>io.github.blad3mak3r.cardio</groupId>
    <artifactId>cardio-postgres</artifactId>
    <version>VERSION</version>
</dependency>
```
</CodeGroup>

<Note>
  Replace `VERSION` with the latest release version.
  
  Current version: ![Maven Central](https://maven-badges.sml.io/sonatype-central/io.github.blad3mak3r.cardio/cardio-postgres/badge.svg)
</Note>

## Verify Installation

After adding the dependency, sync your project and verify the installation by importing the Cardio class:

```kotlin
import io.github.blad3mak3r.cardio.postgres.Cardio
import io.github.blad3mak3r.cardio.postgres.CardioRepository
```

If the imports resolve successfully, you're ready to start using Cardio!
